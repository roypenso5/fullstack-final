# Recipe Box HQ

Recipe Box HQ is a digital repository designed for creating, storing, and organizing personal cooking recipes. 

## Core Capabilities

* **Recipe Registration:** Users can add new recipes by inputting a title, selecting a category, adding an image link, and generating list items for ingredients and step-by-step instructions.
* **Dynamic Database Storage:** All submitted recipes are instantly uploaded to a cloud database, ensuring information is saved securely and persists across sessions.
* **Keyword Filtering:** Users can search their collection in real-time by typing specific ingredients or filtering recipes by category via dropdown controls.
* **Collection Maintenance:** Each recipe card features a dedicated removal control, enabling users to delete recipes from the interface and database simultaneously.
